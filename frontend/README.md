# ✈️ AeroX Enterprise | Next-Gen Flight Booking Ecosystem

A high-performance, enterprise-grade flight search and booking engine built to challenge industry standards. AeroX combines a premium, fluid UI inspired by market leaders with a robust, secure FastAPI backend.

---

## 🚀 Key Features
- **Visual Seat Engine:** Interactive seat mapping with real-time price calculation.
- **Dynamic Check-In:** Full web check-in flow with automated **QR Code Boarding Pass** generation.
- **Real-time Documentation:** Instant PDF E-Ticket generation using `jsPDF`.
- **Enterprise Security:** Secure JWT-based authentication and protected API routes.
- **Premium UX:** Skeleton loading states, glassmorphic components, and Framer Motion animations.

---

## 🛠️ Tech Stack
**Frontend:**
- React 18 (Vite)
- Tailwind CSS (Premium Fluid Design)
- Framer Motion (Micro-interactions)
- Lucide React (Enterprise Iconography)

**Backend:**
- FastAPI (Python)
- SQLAlchemy (ORM)
- SQLite (Production-ready Database)
- JOSE (JWT Implementation)

## ⚙️ Local Setup

### Backend
1. Navigate to directory: `cd backend`
2. Install dependencies: `pip install -r requirements.txt`
3. Run server: `uvicorn main:app --reload`

### Frontend
1. Navigate to directory: `cd frontend`
2. Install packages: `npm install`
3. Launch app: `npm run dev`
